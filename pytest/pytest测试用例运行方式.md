# 运行方式

1. 主函数模式
   1. 运行所有测试用例：pytest.main()
   2. 运行指定模块：pytest.main(['vs','test_login.py'])
2. 命令行模式

参数详解：
+ -s :表示输出调试信息，包括print打印的信息
+ -v :显示更详细的信息，包括模块名，方法名
